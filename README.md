## API Reference

#### Get all items

```http
  GET /api/items
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. 14951c93f3d11e8ac8bed96dd90e8bc7 |

#### Get item

```http
  GET /api/items/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. https://api.openweathermap.org/data/2.5/weather?q=${city}&appid=${apiKey}&units=metric |
